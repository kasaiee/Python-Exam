## سوال اول

خروجی کد زیر چیست؟

`print(type([1,2]))`

1. `<class 'tuple'>`
2. `<class 'int'>`
3. `<class 'set'>`
4. `<class 'complex'>`
5. `<class 'list'>`

5. `<class 'type'>`
